Arduino MEGA 2560

It is the largest of all the boards we talked about and is used in large applications and projects that require a large number of poles and need a larger memory size than the previous types. Among its most important specifications are:
Type of controller: ATMEGA2560
Number of input and output poles: 54
Number of PWM signal poles: 15
Number of analog poles: 16
Processor speed: 16MHz
Working voltage: 5V
Flash program memory size: 256KB
SRAM data memory size: 8KB
EEPROM programmable read-write memory size: 4KB
